# Beta CRM

This is a CRM (Customer Relationship Management) application built with a backend and frontend architecture.

## Project Structure

- **backend/** - Server-side application
- **client/** - Client-side application with HTML, CSS, and SCSS

## Backend

The backend is built with Node.js and includes:
- Controllers for handling business logic
- Database integration
- Models for data structures
- Middleware for request handling
- RESTful API endpoints

## Frontend

The frontend includes:
- Dashboard views
- Kanban board interface
- Task management UI
- Custom styling with SCSS

## Getting Started

### Backend Setup

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure environment variables:
   - Copy `.env.example` to `.env`
   - Update the values as needed

4. Start the server:
   ```bash
   node server.js
   ```

### Frontend

The frontend uses static HTML files with compiled CSS. Use a tool like Prepros or compile SCSS manually.

## Development
